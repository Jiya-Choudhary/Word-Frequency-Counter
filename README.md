# 📈 Word Frequency Counter

A Python-based text analysis tool that counts word occurrences and identifies the most frequently used words in a given text.

## ✨ Features

* 📝 Text Cleaning & Normalization
* 🔢 Word Frequency Analysis
* 📊 Dictionary-Based Counting
* 🔍 Top 5 Most Frequent Words Detection
* ⚡ Efficient Text Processing

## 🛠️ Tech Stack

* Python 3

## 🚀 How to Run

```bash
python word_frequency_counter.py
```

## 📌 Example Output

```text
Enter Text:
Python is great. Python is easy to learn. Python is powerful.

Frequency of Words using Dictionary

{
'python': 3,
'is': 3,
'great': 1,
'easy': 1,
'to': 1,
'learn': 1,
'powerful': 1
}

Top 5 most Frequent words:

{
'python': 3,
'is': 3,
'great': 1,
'easy': 1,
'to': 1
}
```
